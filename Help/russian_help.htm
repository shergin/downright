downright blackrain. справочная система
valentin shergin research laboratory
Общие сведения
Назначение программы — организация управления компьютером жестами мыши. Жест — это символ, при рисовании которого, обычно правой кнопкой мыши, выполняется некоторая определенная команда. Такая реализация вызова часто употребляемых команд позволяет упростить и ускорить работу на компьютере, сделать ее интуитивно понятной — гораздо проще нарисовать размашистый символ, чем долго искать команду в объемном меню, целиться мышкой по маленьким кнопкам и постоянно переключаться между мышью и клавиатурой.
Программа запускается при загрузке компьютера и постоянно отслеживает движения мыши, распознавая символы и выполняя соответствующие им команды. При распознавании одного символа в жесте последовательно выполняются все команды ему соответствующие. Команды посылаются окну, на фоне которого начали рисовать символ.
Кроме названия, набора символов и команд каждый жест имеет параметры совместимости, определяющие ограничения его вызова на фоне определенных программ. Различают три вида совместимости:
  • совместим со всеми программами
  • совместим со всеми, кроме указанных программ
  • совместим только с указанными программами
Все жесты можно перенастроить или добавить свои, полностью изменив символы, набор команд, ограничения на совместимость и название.
Описание интерфейса
Элементы управления в главном окне сгруппированы по назначению в несколько вкладок.
Вкладка «О программе»
Вкладка содержит краткую справочную информацию.
Вкладка «Жесты»
Позволяет изменять параметры существующих жестов и создавать новые.
Слева размещается древовидный список жестов, справа – вкладки определяющие свойства выбранного жеста. Для удобства работы при большом количестве жестов их можно группировать по папкам. Элементы внутри папок всегда сортируются по алфавиту. Для перемещения жеста между папками просто перетащите его левой кнопкой мыши в нужное место. Для создания или удаления элементов дерева необходимо выбрать соответствующий пункт контекстного меню, вызываемого щелчком правой кнопки мыши.
Вкладки, определяющие свойства жеста:
Вкладка «Описание»
Каждый жест должен иметь информативное название, которое можно задать в соответствующем поле ввода.
Кроме этого в этой вкладке можно назначить дополнительное описание жеста и заблокировать его применение.
Вкладка «Команды»
В данной вкладке необходимо описать последовательность команд, которые будут выполнены при распознавании любого из символов в жесте.
Вверху находится список команд, внизу – область свойств в виде вкладок, где выбранная закладка определяет тип выделенной команды в списке, и соответственно набор возможных для него параметров.
Для добавления, удаления и изменения порядка команд необходимо использовать контекстное меню, вызываемое щелчком правой кнопки мыши.
Команды делятся на несколько типов, соответствующим вкладкам:
  • Простые
  • Запуск
  • Текст
  • Клавиатура
  • Сообщение
Одной или последовательностью команд можно реализовать практически любое действие.
Вкладка «Простые»
Вкладка содержит набор наиболее употребляемых действий с окнами, документами и системой, такие как:
  • Закрыть
  • Развернуть
  • Свернуть
  • Свернуть все (показать рабочий стол)
  • Копировать
  • Вставить
  • Вырезать
  • Открыть
  • Сохранить
  • Печать
  • Вперед
  • Назад
  • Следующее окно
  • Предыдущее окно
  • Отменить
  • Вернуть
  • Выделить все
  • Закрыть документ
  • Выключить компьютер
Вкладка «Запустить»
Позволяет запускать приложения, открывать папки, Интернет ресурсы, вызывать функции dll-библиотек (используйте rundll.exe) и прочее.
Вкладка «Текст»
Позволяет вставлять текст в произвольное окно при распознавании жеста.
Вкладка «Клавиатура»
Позволяет эмулировать нажатие клавиш.
Вкладка «Сообщение»
Позволяет посылать окну произвольные сообщения Windows.
Вкладка «Символы»
Данная вкладка позволяет изменять и добавлять символы. Всё пространство вкладки занимает поле для рисования, в правом нижнем углу которого располагаются изображения уже имеющихся в жесте символов. Для того чтобы добавить новый символ просто нарисуйте его левой кнопкой мыши в поле для рисования. Для удаления символа щелкните по нему правой кнопкой мыши.
Вкладка «Совместимость»
Жест может быть как полностью универсальным, так и специально предназначен для определенной программы, или же быть универсальным, но некорректно работать с некоторыми программами. Для определения параметров совместимости сначала необходимо задать ее тип, выбрав его из списка: «совместим со всеми», «совместим со всеми кроме указанных», «совместим только с указанными». Далее необходимо добавить в список строку, щелчком правой кнопки мыши, вызвав контекстное меню и выбрав там соответствующий пункт. Затем необходимо ввести фрагмент имени окна программы в поле ввода над списком. Одновременно с изменением текста в поле ввода он будет меняться и в списке. Для удаления сток из списка можно нужно использовать тоже контекстное меню.
